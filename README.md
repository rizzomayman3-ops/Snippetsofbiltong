### Setup Instructions

#### To Deploy on Vercel:

Vercel automatically deploys from GitHub. Just push changes to the main branch!

#### To Set Up Custom Domain:

1. Purchase your domain (e.g., snippetsofbiltong.com)
2. Go to your Vercel project settings → Domains
3. Add `snippetsofbiltong.com` as a custom domain
4. Follow Vercel's DNS configuration instructions
5. Point your domain registrar to Vercel's nameservers

### Customization

- **Add Product Images:** Replace `.product-image-placeholder` sections in `index.html` with actual image tags
- **Update Colors:** Edit `#d4af37` (gold) and `#1a1a1a` (dark) in `styles.css`
- **Modify Content:** Edit text directly in `index.html`
- **Add Social Links:** Update URLs in the footer section
